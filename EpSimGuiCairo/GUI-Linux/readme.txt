How to start the GUI:
1. Open the terminal (ctrl + alt + t in Ubuntu)
OR: Right-click in this folder (GUI-Linux) and choose "open in terminal"
2. Enter "./Epidemic-Simulation-GUI" in terminal (without "")

Alternative: (Highly recommended)
Use WSL (Windows-Subsystem for Linux) for better performance!
For instructions see here: https://docs.microsoft.com/de-de/windows/wsl/install-win10
